# MediPredict 🩺

AI-powered disease prediction system that analyzes user symptoms and provides possible diseases, severity level, and recommendations.

## Live Demo

🚀 Try the deployed app here:

https://medipredict-ai.streamlit.app/

## Features

* User Registration and Login
* Symptom-based Disease Prediction
* AI-powered Medical Recommendations
* Disease Confidence Scores
* Severity Analysis
* Risk Level Assessment
* Prediction History Tracking
* PDF Report Generation

## Technologies Used

* Python
* Streamlit
* Groq API
* SQLite
* Pandas
* Bcrypt
* FPDF

## How to Run

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create Environment File

Create a `.env` file in the project root:

```env
GROQ_API_KEY=your_groq_api_key
```

### Run the Application

```bash
streamlit run app.py
```

## Database

The application uses SQLite database.

### Users Table

| Field    | Type    |
| -------- | ------- |
| id       | INTEGER |
| username | TEXT    |
| password | TEXT    |

### History Table

| Field    | Type      |
| -------- | --------- |
| id       | INTEGER   |
| username | TEXT      |
| symptoms | TEXT      |
| disease  | TEXT      |
| date     | TIMESTAMP |
